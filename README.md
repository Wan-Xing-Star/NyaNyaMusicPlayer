这个播放器是自用的,没有任何技术含量 =w= 

第一次启动后要先配置设置文件,即config.json (该文件没有的话运行一次自动生成)

设置:
{
    "音乐文件夹路径": [],   //这里放音乐文件夹父目录,可以放多个(用/代替\或者使用\\) //ps:"音乐文件夹路径": ["F:/music","Z:\\music"],
    "播放模式": 0, // 0:顺序播放; 1:伪随机; 2:真随机;
    "定时播放": 0, //单位:分钟  //0就是不开启
    "播完暂停": true,   //如果定时播放时间到了,而有一首歌正在播放,是播完暂停还是立即结束
    "平衡播放": true,   //在伪随机和列表播放时,避免个别歌曲重复次数过多
    "平衡播放限度": 0,    //设置一首歌曲播放次数在平均播放次数的范围值
    "循环播放不计入平衡播放": true, //循环播放的次数将不被用于平衡播放的判断
    "伪随机可重复曲数": -1,   //伪随机允许生成的可重复歌曲数量  -1为随歌单动态生成
    "按键管理": {
        "暂停": "<ctrl>+<alt>+p",
        "退出": "<ctrl>+<alt>+s",
        "音量+": "<ctrl>+<alt>+<up>",
        "音量-": "<ctrl>+<alt>+<down>",
        "上一曲": "<ctrl>+<alt>+<left>",
        "下一曲": "<ctrl>+<alt>+<right>",
        "单曲循环": "<ctrl>+<alt>+l"    //按下会单曲循环你按下时播放的歌曲  //再按一次接触单曲循环状态,继续按设定播放
    },
    "音量变化值": 0.05,  //调节音量值 //范围0.00~1.00
    "音量初始值": 0.8,    //初始音量 //范围0.00~1.00
    "日志等级": 3  //日志记录详细度  //范围4~0
}


关于平衡播放
    歌曲    |    播放次数   |循环播放次数
    1.flac |          4   |         2
    2.flac |          7   |         4
    3.flac |          1   |         0

    //开启循环播放不计入平衡播放
    那么3首歌的平均播放次数为(4-2)+(7-4)+(1-0)=2,
    播放1.flac,平衡值为(4-2)=2,可播放
    播放2.flac,平衡值为(7-4)=3,其播放次数超过平均值,将跳过,不再播放

    若是设置限度为3
    播放2.flac,其判断条件为 3 <= (ave + 3) 为 T ,即不大于允许值,可以播放

    //关闭循环播放不计入平衡播放
    那么3首歌关于平衡值的计算将不会减去循环播放次数
