这个播放器是自用的,没有任何技术含量 =w= 

第一次启动后要先配置设置文件,即config.json (该文件没有的话运行一次自动生成
设置:
{
    "音乐文件夹路径": [],   //这里放音乐文件夹父目录,可以放多个(用/代替\或者使用\\) //ps:"音乐文件夹路径": ["F:/music","Z:\\music"],
    "随机播放": [
        true,       //第一个是启用随机播放,第二个是真假随机  //如[true,false]是伪随机,[true,true]是真随机(经常有重复歌曲),[false,true]和[false,false]是列表播放
        false
    ],
    "定时播放": 0, //单位:分钟  //0就是不开启
    "播完暂停": true,   //如果定时播放时间到了,而有一首歌正在播放,是播完暂停还是立即结束
    "平衡播放": True,   //在伪随机和列表播放时,避免个别歌曲重复次数过多
    "平衡播放限度": 1,    //设置一首歌曲播放次数在平均播放次数的范围值
    "伪随机可重复曲数": -1,   //伪随机允许生成的可重复歌曲数量  -1为随歌单动态生成
    "按键管理": {
        "暂停": "<ctrl>+<alt>+p",
        "退出": "<ctrl>+<alt>+s",
        "音量+": "<ctrl>+<alt>+<up>",
        "音量-": "<ctrl>+<alt>+<down>",
        "上一曲": "<ctrl>+<alt>+<left>",
        "下一曲": "<ctrl>+<alt>+<right>",
        "单曲循环": "<ctrl>+<alt>+l"    //按下会单曲循环你按下时播放的歌曲  //再按一次接触单曲循环状态,继续按设定播放
    },
    "音量变化值": 0.05,  //调节音量值 //范围0.00~1.00
    "音量初始值": 1.0,    //初始音量
    "日志等级": 3  //日志记录详细度  //范围4~0
}


关于平衡播放
    歌曲    |    播放次数
    1.flac |          0
    2.flac |          5
    3.flac |          1

    那么3首歌的平均播放次数为2,
    播放1.flac,可播放
    播放2.flac,其播放次数超过平均值,将跳过,不再播放

    若是设置限度为3
    播放2.flac,其判断条件为 5 < (ave + 3) 为F,即不大于允许值,可以播放
