这个播放器是自用的,没有任何技术含量 =w= 

运行内存占用<=10MB  cpu占用可以忽略
歌曲统计显示是直接time//3600的

第一次启动后要先配置设置文件,即config.json (该文件没有的话运行一次自动生成
设置:
{
    "音乐文件夹路径": [],   //这里放音乐文件夹父目录,可以放多个(用/代替\或者使用\\) //ps:"音乐文件夹路径": ["F:/music","Z:\\music"],
    "随机播放": [
        true,       //第一个是启用随机播放,第二个是真假随机  //如[true,false]是伪随机,[true,true]是真随机(经常有重复歌曲),[false,true]和[false,false]是列表播放
        false
    ],
    "定时播放": 0, //单位:分钟  //0就是不开启
    "播完暂停": true,   //如果定时播放时间到了,而有一首歌正在播放,是播完暂停还是立即结束
    "按键管理": {
        "暂停": "<ctrl>+<alt>+p",
        "退出": "<ctrl>+<alt>+s",
        "音量+": "<ctrl>+<alt>+<up>",
        "音量-": "<ctrl>+<alt>+<down>",
        "上一曲": "<ctrl>+<alt>+<left>",
        "下一曲": "<ctrl>+<alt>+<right>",
        "单曲循环": "<ctrl>+<alt>+l"    //按下会单曲循环你按下时播放的歌曲  //再按一次接触单曲循环状态,继续按设定播放
    },
    "音量变化值": 0.05,  //调节音量值 //范围0.00~1.00
    "音量初始值": 1.0,    //初始音量
    "日志等级": 3  //日志记录详细度  //范围4~0
}


虽然数据文件有主数据和每日数据,但实际上每日数据根本就没有用到,不过不用担心,每日数据就算没用存10年才不过5mb

NyaData.exe是查看统计数据的,代码是修改ai的
